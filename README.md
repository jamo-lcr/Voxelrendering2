A simple voxelrenderer, that Renders a heightmap

## Technologies

- OpenTK
- C#

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jamo-lcr/Voxelrendering2.git
2.Navigate to the project directory
  cd Voxelrendering2
3.Add OpenTK
  dotnet add package OpenTK
  dotnet restore
4.Build and Run 
  dotnet build
  dotnet run
